**Brief Project Description**
This project is a Student Profile Management System developed for the CSC584 course. The main goal of this system is to allow users to manage student profiles using a website connected to a database.

It handles all basic database actions: adding new records, showing them in a table, updating existing details, and deleting records (CRUD operations). The system uses HTML and Bootstrap 5 for a clean, user-friendly design, and uses JSP, Servlets, and an Apache Derby Database on the backend to securely process and store the student information.

**List of Implemented Features**
**1. Add Student Profile (index.html and profile.jsp)**

A simple registration form where you can type in a Student ID, Name, Programme, Email, Hobbies, and a Self-Introduction.

Once submitted, it saves the data into the database and shows a success confirmation screen.

**2. View All Profiles (viewProfiles.jsp)**

A clean dashboard table that displays all the registered students currently saved in the database so you can view everyone's profile in one place.

**3. Search Profiles (search.jsp)**

A dedicated search page where you can type a Student ID or a Name to instantly filter and find specific students. It supports partial matching (e.g., typing just a part of a name will still find them).

**4. Edit Profile (edit.jsp)**

Clicking the "Edit" button opens a form that looks just like the registration form, but it is already pre-filled with that student's current data.

You can update any information (like changing a programme code to CS259 or CDCS267) while the Student ID stays locked to prevent mistakes.

**5. Delete Profile (delete.jsp)**

Clicking the "Delete" button will remove that student's profile from the database completely.

It includes a pop-up warning message ("Are you sure you want to delete this profile?") so you do not accidentally delete a record by mistake.
